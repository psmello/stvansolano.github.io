#XamarinDiplomado

¡Participa en el #XamarinDiplomado Latinoamérica! Durante el mes de Octubre y Noviembre de 2016 estaré realizando el primer diplomado para Latinoamérica en conjunto con Microsoft y los Xamarin MVPs Enrique Aguilar y Humberto Jaimes a través de un diplomado en línea 100% en español a lo largo de 5 semanas y orientado a la audiencia técnica Microsoft en Latinoamérica.


¡Aun puedes inscribirte en el #XamarinDiplomado en español! Impartido por @stvansolano @EnriqueAguilar @HJaimesDev https://t.co/aQ6nA312TZ

— Miguel de Icaza (@migueldeicaza) 18 de noviembre de 2016

El diplomado cubrirá los siguientes temas:

Módulo 1: Introducción al desarrollo móvil

Lección 1: Introducción a Xamarin.iOS
Lección 2: Introducción a Xamarin.Android
Lección 3: Introducción a Xamarin.Forms
Lección 4: Portabilidad de código .Net (Shared Code / PCL)

Módulo 2: Xamarin Android

Los temas abarcan este módulo son:

Lección 1: Estructura de una aplicación Android con Xamarin
Lección 2: Trabajando con el Diseñador (AXML)
Lección 3: Introducción a los Activities
Lección 4: Controles de UI
Lección 5: Introducción al Acceso de Datos SQLite y Servicios Web con Xamarin.Android

Módulo 3: Xamarin iOS

Los temas abarcan este módulo son:

Lección 1: Estructura de una aplicación iOS con Xamarin
Lección 2: Trabajando con el Diseñador (XIB / StoryBoards)
Lección 3: Introducción a los eventos y delegados
Lección 4: Introducción a los Controles de UI

Módulo 4: Xamarin Forms

Los temas abarcan este módulo son:

Lección 1: Introducción a la Estructura de Xamarin Forms
Lección 2: Modelo-Vista-Vista de Modelo (MVVM)
Lección 3: La Estructura del Lenguaje XAML
Lección 4: Almacenamiento SQLite en Xamarin.Forms
Lección 5: Consumo de Servicios REST en Xamarin.Forms

Módulo 5: Servicios de nube para tu App Xamarin

Los temas que forman parte de este módulo son:

Lección 1: Introducción a Azure App Service
Lección 1: Introducción a Azure Mobile Apps
Lección 3: Sincronización de datos fuera de línea
Lección 4: Servicio de notificaciones para tu App
